# Quiz Microservices Architecture

##  Project Overview
This project is a Microservices-based Quiz Application built using Spring Boot and Spring Cloud.

It includes service discovery, API gateway, and independent services for quiz and questions.

---

##  Architecture

- Service Registry (Eureka Server)
- API Gateway
- Quiz Service
- Question Service

---

##  Tech Stack

- Java
- Spring Boot
- Spring Cloud
- Eureka Server
- API Gateway
- Maven
- REST APIs

---

##  Microservices Structure

Quiz-microservice/
│
├── service-registry  
├── api-gateway  
├── quiz-service  
└── question-service  

---

##  How to Run

1. Start Service Registry
2. Start Question Service
3. Start Quiz Service
4. Start API Gateway
5. Access APIs via Gateway

---

##  Features

- Service Discovery using Eureka
- API Gateway Routing
- REST-based communication
- Independent deployment of services
